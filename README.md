# Weather-Diary1

## Автор
Обучающийся

## Описание
Приложение для ведения дневника погоды. Пользователь может сохранять записи о погоде за разные дни и просматривать историю.

## Функции
- Добавление записей о погоде
- Указание даты, температуры и описания
- Просмотр сохранённых записей
- Сохранение данных в JSON
- Загрузка данных из JSON

## Требования
- Python 3.x

## Запуск
1. Установить Python
2. Открыть проект
3. Запустить:

   python main.py

## Примечание
Данные сохраняются в JSON-файл автоматически.
